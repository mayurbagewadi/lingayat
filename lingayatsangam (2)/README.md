# LingayatSangam - Premium Matrimonial Platform

LingayatSangam is a modern, culturally attuned matrimonial platform dedicated to the Lingayat community. Blending tradition with technology, it offers a dignified and elegant space for finding life partners.

## 🌟 Features

- **Royal Aesthetic**: Designed with a premium "Royal Purple & Gold" theme reflecting tradition and luxury.
- **Community Focused**: Tailored fields for Lingayat sub-castes (Panchamasali, Banajiga, Sadar, etc.).
- **Smart Navigation**: Seamless transitions between Landing, Login, and Registration views without page reloads.
- **AI Integration**: Powered by Google Gemini AI (Backend ready) for smart profile bio generation (Feature currently hidden for MVP).
- **Responsive Design**: Fully responsive UI built with Tailwind CSS.
- **Fluid Animations**: Smooth interactions using Framer Motion and GSAP.

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **Icons**: Lucide React
- **AI**: Google GenAI SDK (Gemini 2.5 Flash)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/lingayat-sangam.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file and add your Google Gemini API Key:
   ```env
   API_KEY=your_google_genai_api_key
   ```

4. **Run the development server**
   ```bash
   npm start
   ```

## 📂 Project Structure

- `components/`: Reusable UI components (Hero, Navbar, Features, Forms).
- `services/`: API integrations (Gemini AI service).
- `types.ts`: TypeScript interfaces for type safety.
- `index.html`: Entry point with Tailwind configuration.

## 🎨 Theme

The application uses a custom Tailwind configuration:
- **Primary**: Royal Purple (`#7c3aed` - `#2e1065`)
- **Accent**: Gold (`#fbbf24` - `#451a03`)
- **Fonts**: 'Playfair Display' (Serif) and 'Lato' (Sans-serif)
